# Semantic Ontology v0.1

最小文明语义集合（Minimal Civilization Ontology）。  
Ontology 是 **梦境与现实之间的翻译器**：Semantic Dream Layer → interpretation → Structured World。

## 三种核心对象

| 类型 | 含义 | 文件 |
|------|------|------|
| **Entity** | 世界中的存在 | [entities.json](entities.json) |
| **Relation** | 存在之间的关系 | [relations.json](relations.json) |
| **Action** | 可执行的行为 | [actions.json](actions.json) |

## 设计原则

1. **极简** — 只定义必须存在的概念  
2. **可扩展** — 未来可增加 economy, culture, law, technology 等  
3. **可执行** — 驱动 agent decision、simulation、governance

# RFC-0001: Datong Digital Civilization Concept

**Status:** Draft  
**Author:** OpenDatong  
**Date:** 2026-03-13

## Summary

This RFC proposes the idea of building a semantic-based digital society inspired by the philosophical concept of Datong.

## Motivation

Human conflicts often arise from semantic ambiguity and misaligned incentives. If agents communicate through shared semantic ontology, trust and cooperation may increase over time.

## Proposal

Create a simulation environment where:

- Agents communicate through shared semantic ontology
- Governance mechanisms can emerge from interaction
- Cooperation and conflict can be observed and analyzed

## Open Questions

- How to define value ontology?
- How should governance emerge?
- Should the world be simulated visually or textually?

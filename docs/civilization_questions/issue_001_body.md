This project explores the construction of a **digital civilization**.

The first question is fundamental:

---

## What is the nature of the digital world we are building?

### Option A: LLM Dream Space

The digital world is primarily a **semantic space** generated by LLM reasoning and imagination.

- LLM reasoning driven
- fluid reality
- semantic imagination
- Like: *collective dream space* (人类语言文明的集体梦)

### Option B: Structured Digital System

The digital world is a **structured system** defined by rules, protocols, and persistent state.

- ontology
- rules
- protocols
- persistent state
- Like: *digital operating system*

### Option C: Hybrid Model

Dream Layer + Structure Layer (LLM imagination → semantic interpretation → structured execution).

# 重要决定：Ontology v0.1 价值系统（Values）

Ontology v0.1 定义了 **Entity**、**Relation**、**Action**，但还缺少一个关键部分：**价值（Values）**。

没有价值体系，Agent 的行为就没有方向。价值将影响：
- agent decision
- governance style
- social evolution

---

## 请选择或讨论：价值系统的文化基因

### 方案 A：儒家价值体系

例如：
- 仁 (benevolence)
- 义 (righteousness)
- 礼 (propriety)
- 智 (wisdom)
- 信 (trustworthiness)

灵感来自 Confucius，与「大同」理念同源。

### 方案 B：普适文明价值

例如：
- fairness
- cooperation
- freedom
- trust
- knowledge

更偏现代普世价值表述。

这一选择将 **直接决定数码文明的文化基因**。欢迎在评论中投票（如 "A" / "B"）或提出混合、修正方案。
